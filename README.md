# 🌐 나만의 포트폴리오 웹사이트 제작 과제

이 프로젝트는 나를 소개하고 그동안의 작업물을 한눈에 볼 수 있도록 제작한 **개인 웹 포트폴리오** 저장소입니다.

## 🚀 프로젝트 소개
* **목적:** HTML/CSS를 활용한 웹페이지 제작 및 GitHub Pages를 통한 배포 실습
* **주요 내용:** 자기소개, 기술 스택 아카이빙, 프로젝트 경험 정리

## 🛠️ 사용한 기술 (Tech Stacks)
* HTML5
* CSS3
* GitHub Pages (웹 호스팅)

## 🔗 웹사이트 링크
* 배포된 웹사이트는 다음 링크에서 확인하실 수 있습니다: [여기에 배포된 내 GitHub Pages 주소 적기]
